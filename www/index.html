Sub-Etha is a sophisticated mailing list manager, similar in many respects to the popular Mailman package. Sub-Etha's features include:

  • A user-friendly web interface
  • Virtual domains (ie list@foo.com and list@bar.com are separate lists)
  • Intelligent VERP bounce processing
  • Searchable, threaded archives
  • Clusterable for nearly unlimited scalability
  • Easy integration with any MTA
  • EJB and SOAP interfaces for automation

Sub-Etha is a three-tiered J2EE application using EJB3 and JMS. It is designed to deploy on JBoss and relies on numerous other open source projects including Hibernate, Tagonist, and Lucene.

This project is still in the design phase. Its authors are:

  • Jeff Schnitzer
  • Scott Hernandez
  • Ian McFarland
  • Jon Stevens

There will be two distributions. The first, for advanced administrators, is an EAR file which can be deployed into any JBoss instance. The second, for novices, is a "just-unzip-and-run" package containing standalone JBoss with Sub-Etha preconfgured and all unnecessary services removed.
